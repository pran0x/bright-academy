# Bright Academic Care Website

## Overview
The Bright Academic Care website is designed to provide information about the coaching center, including its courses, faculty, testimonials, and admission process. The website aims to create an engaging and informative experience for prospective students and their parents.

## Features
- **Responsive Design**: The website is designed to be mobile-friendly and responsive across various devices.
- **Interactive Elements**: Includes a hero slider, testimonials slider, and an interactive admission form.
- **Gallery Section**: Showcases images of the institution's activities and environment.
- **Google Form Integration**: Allows users to submit admission applications online.

## Setup Instructions
1. **Clone the Repository**: 
   ```bash
   git clone <repository-url>
   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd bright-academic-care-website
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Open the Website**: Open `src/index.html` in a web browser to view the website.

## Usage
- Navigate through the different sections using the navigation menu.
- Fill out the admission form to apply online.
- View testimonials and the gallery to learn more about the institution.
